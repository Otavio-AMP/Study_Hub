# PT/BR: Study Hub - Aplicação web para organização acadêmica.
## Introdução 
Study Hub é uma aplicação web gratuita voltada para a organização de assuntos relacionados à estudos.

**Possui 3 páginas principais:**
* Uma para criação e edição de anotações, inspirada em aplicativos como o Notion.
* Uma com um calendário, onde o usuário pode adicionar eventos (provas, trabalhos, aulas e etc), e relacioná-los com suas anotações.
* A terceira uma páguina com todas as avaliações feitas e à fazer, com a nota de cada avaliação, o total e a média, e o necessário para ser aprovado.
## Desenvolvimento
**Frontend**
* HTML e CSS
* JavaScript

**Backend**
* Python 3.14
    * Framework Django

Criação de conta para armazenamento de dados em nuvem.
Completamente Open-Source e gratuito.
Projeto realizado para atividade avaliativa A3, para faculdade UNA - Campus Betim-MG.

# EN/USA: Study Hub - Web Application for Academic Organization.
## Introduction
Study Hub is a free web application focused on organizing study-related matters.

**It has 3 main pages:**
* One for creating and editing notes, inspired by applications like Notion.
* One with a calendar, where the user can add events (tests, assignments, classes, etc.) and link them to their notes.
* The third page shows all completed and upcoming assessments, with the grade for each one, the total and average, and what is needed to pass.

## Development
**Frontend**
* HTML and CSS
* JavaScript

**Backend**
* Python - Django framework

Completely Open-Source and free.
Project developed for an A3 assessment activity for UNA University - Betim-MG Campus.
